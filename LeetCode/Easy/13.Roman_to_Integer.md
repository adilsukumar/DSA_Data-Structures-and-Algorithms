# LeetCode 13. Roman to Integer

## Problem Statement

Given a Roman numeral string `s`, convert it to an integer and return its corresponding value.

Roman numerals are represented by seven different symbols:

| Symbol | Value |
| ------ | ----- |
| I      | 1     |
| V      | 5     |
| X      | 10    |
| L      | 50    |
| C      | 100   |
| D      | 500   |
| M      | 1000  |

Special subtractive combinations include:

* IV = 4
* IX = 9
* XL = 40
* XC = 90
* CD = 400
* CM = 900

---

## Approach

This solution traverses the Roman numeral string from left to right.

### Algorithm

1. Check for the six special subtractive combinations (`IV`, `IX`, `XL`, `XC`, `CD`, `CM`).
2. If a special pair is found:

   * Add its corresponding value to the sum.
   * Skip the next character since it has already been processed.
3. Otherwise, add the value of the current Roman numeral character.
4. Continue until the entire string has been processed.

This approach correctly handles both standard Roman numerals and subtractive notation.

---

## Complexity Analysis

| Metric           | Complexity |
| ---------------- | ---------- |
| Time Complexity  | **O(n)**   |
| Space Complexity | **O(1)**   |

### Explanation

* Each character is processed at most once.
* Only a few integer variables are used, regardless of input size.

---

## Solution

```cpp
class Solution {
public:
    int romanToInt(string s) {
        int sum, I, V, X, L, C, D, M;
        sum = 0;
        I = 1;
        V = 5;
        X = 10;
        L = 50;
        C = 100;
        D = 500;
        M = 1000;

        for(int i = 0; i < s.size(); i++) {
            if (s[i] == 'I' && s[i+1] == 'V') {
                sum += 4;
                i++;
            }
            else if (s[i] == 'I' && s[i+1] == 'X') {
                sum += 9;
                i++;
            }
            else if (s[i] == 'X' && s[i+1] == 'L') {
                sum += 40;
                i++;
            }
            else if (s[i] == 'X' && s[i+1] == 'C') {
                sum += 90;
                i++;
            }
            else if (s[i] == 'C' && s[i+1] == 'D') {
                sum += 400;
                i++;
            }
            else if (s[i] == 'C' && s[i+1] == 'M') {
                sum += 900;
                i++;
            }
            else if (s[i] == 'I') {
                sum += 1;
            }
            else if (s[i] == 'V') {
                sum += 5;
            }
            else if (s[i] == 'X') {
                sum += 10;
            }
            else if (s[i] == 'L') {
                sum += 50;
            }
            else if (s[i] == 'C') {
                sum += 100;
            }
            else if (s[i] == 'D') {
                sum += 500;
            }
            else if (s[i] == 'M') {
                sum += 1000;
            }
        }
        return sum;
    }
};
```
