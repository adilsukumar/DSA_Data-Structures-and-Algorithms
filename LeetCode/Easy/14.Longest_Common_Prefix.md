# LeetCode 14. Longest Common Prefix

## Problem Statement

Write a function to find the longest common prefix string amongst an array of strings.

If there is no common prefix, return an empty string `""`.

---

## Approach

This solution uses a **vertical scanning approach**.

### Algorithm

1. Use the first string as the reference string.
2. Traverse each character of the first string.
3. For every character position:

   * Compare it with the character at the same position in all other strings.
   * If any string is shorter than the current position, return the prefix found so far.
   * If a mismatch is found, return the prefix found so far.
4. If all strings match at the current position, add the character to the answer.
5. Continue until the end of the first string is reached.

Since all strings are checked column by column, the algorithm efficiently finds the longest common prefix.

---

## Complexity Analysis

| Metric           | Complexity                           |
| ---------------- | ------------------------------------ |
| Time Complexity  | **O(n × m)**                         |
| Space Complexity | **O(1)** *(excluding output string)* |

### Explanation

* `n` = number of strings.
* `m` = length of the shortest common prefix examined.
* Each character position is compared across all strings.

---

## Solution

```python
class Solution:
    def longestCommonPrefix(self, strs: List[str]) -> str:
        ans = ""

        for i in range(len(strs[0])):
            for j in range(len(strs)):
                if i >= len(strs[j]):
                    return ans

                if strs[j][i] != strs[0][i]:
                    return ans

            ans += strs[0][i]

        return ans
```

## Key Takeaway

Using the first string as a reference and comparing characters position-by-position across all strings provides a simple and efficient way to find the longest common prefix. The solution stops immediately when a mismatch is encountered, avoiding unnecessary comparisons.
